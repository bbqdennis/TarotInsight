1. App 名字：
【Tarot Insight 解牌師】

2. App 目的：
【透過用戶的提問，自動選擇最合適的塔羅牌陣，並引導抽牌，最後根據每張牌在牌陣中的位置，結合牌義進行個性化解說，協助用戶釐清問題、找到方向。】

3. App 功能與業務邏輯：
【以下每個功能會配對其對應邏輯】

功能 1：解析用戶提問，推薦最適合的塔羅牌陣  
業務邏輯：系統先讀取「塔羅牌牌陣大全目錄」，根據 NLP 分析用戶問題的主題（感情、工作、健康、自我成長等），匹配相應類型的牌陣（例如：三張牌、十字架、關係解析陣等）。

功能 2：讀取並顯示所選牌陣的說明與排法  
業務邏輯：系統從資料庫中讀取所選牌陣的排法圖與說明，展示每個位置代表的意義（例如：過去 / 現在 / 未來），並預備抽牌介面。

功能 3：引導用戶抽牌或提供自動抽牌選項  
業務邏輯：用戶可手動從牌堆中一張一張地抽取，或點選「自動抽牌」，系統隨機抽出對應張數的塔羅牌（讀取指定塔羅牌圖片與牌義）。

功能 4：解說每張牌在牌陣中代表的意義  
業務邏輯：每張牌結合其正位 / 逆位情況，根據所在位置的含義與牌義，提供一段清楚易懂的說明，並可選擇展開詳細分析。

功能 5：提供完整解牌報告匯出  
業務邏輯：系統將使用的牌陣、每張牌的圖片、位置與解釋整合成一份報告，可供用戶儲存、分享或列印。

4. App 頁面與功能元素：

頁面 1：提問與牌陣推薦頁  
- 功能：
  - 用戶輸入提問
  - 顯示推薦牌陣
- UI 元素：
  - 問題輸入框
  - 「送出問題」按鈕
  - 推薦牌陣卡片（含名稱、用途簡介、選擇按鈕）

頁面 2：牌陣說明與抽牌介面  
- 功能：
  - 顯示牌陣圖與說明
  - 抽牌（手動 / 自動）
- UI 元素：
  - 牌陣示意圖
  - 每個位置標註含義
  - 牌堆圖像（可互動抽牌）
  - 自動抽牌按鈕

頁面 3：解牌頁  
- 功能：
  - 顯示每張牌與解釋
  - 點擊展開詳解
- UI 元素：
  - 牌卡圖像（每張顯示位置）
  - 精簡解釋文字區塊
  - 「查看更多」按鈕展開詳細牌義分析

頁面 4：解牌報告頁  
- 功能：
  - 匯總所有解牌內容
  - 提供下載 / 儲存 / 分享
- UI 元素：
  - 報告預覽區塊
  - 下載 PDF 按鈕
  - 分享至 LINE / Email 等按鈕
